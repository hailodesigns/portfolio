Spectral Oracle

Spectral Oracle

Spectral Oracle was a project created to show how the use of animation increases user experience. When the user scrolls to a different section of the website the animation loads at a rate that is comfortable for the user to take in the information being presented.

Crypdex

Crypdex

Crypdex is a database used to collect the information of different cryptid creatures. This project was created to show the basics of OnClick JQuery functions and how they can be used to increase user experience and load information.

Out of the Wild

Out of the Wild

Out of the Wild is an exotic and wildlife animal rescue that focuses on rehabilitation and rehoming of exotic animals. This website was created to show the use of local servers and how you can load different HTML files into different sections based on what the user clicks on. I implemented this technique on my portfolio to further experiment with it.
